# TP3 - Buscador de Libros

Es Aplicación Android que permite buscar libros de una lista predefinida y ver sus detalles.

## Características

- **Dos Activities**: MainActivity (búsqueda) y DetailActivity (detalles)
- **Arquitectura MVVM**: Separación de responsabilidades con ViewModel
- **View Binding**: Acceso type-safe a las vistas
- **LiveData**: Observación reactiva de datos
- **RecyclerView + Adapter**: Lista de sugerencias dinámica

## Funcionamiento

1. En la pantalla principal, comenzá a escribir el título de un libro
2. Aparecerán sugerencias automáticamente debajo del campo de búsqueda
3. Tocá una sugerencia o presioná "Buscar" para ir al detalle
4. Se muestra la segunda pantalla con los detalles del libro
